1. **Clone the repository**

   ```bash
   git clone <repo_url>
   cd calculator_app
   ```

2. **Create a virtual environment** (optional but recommended)

   
   python -m venv venv
   source venv/bin/activate   # For Mac/Linux

   python manage.py runserver
